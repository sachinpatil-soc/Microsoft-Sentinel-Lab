# 📝 Lessons Learned

## 💻 Technical Lessons

- 🌐 Internet-facing systems receive automated attacks within hours.
- 🔍 KQL enables rapid filtering and investigation of security events.
- 🗺️ Geo-IP enrichment provides useful attack context.
- 🛡️ Microsoft Sentinel simplifies centralized monitoring.

---

## ⚙️ Operational Lessons

- 📄 Documentation is as important as technical investigation.
- 📋 Standardized playbooks improve analyst consistency.
- ⏳ Proper alert triage reduces investigation time.
- 🚨 Escalation criteria should be clearly defined.
